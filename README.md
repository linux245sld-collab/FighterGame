# FighterGame
這個是一個終端小遊戲，經典的戰鬥機遊戲，相信是大家的童年小遊戲，本次本人以命令列的形式還原，即使在沒有任何桌面環境的情況下也可以遊玩
玩法:方向鍵控制飛機行動，空格射擊
使用方法：首先需要安裝java,接著在Release裡面下載class檔案，在終端機內輸入java FighterGame即可啟動遊戲
支援的作業系統：Linux,macOS，FreeBSD,OpenBSD,NetBSD以及其他Unix-like系統

これはターミナルゲームです。多くの人が子供の頃に遊んだであろう、古典的な戦闘機ゲームです。このバージョンはコマンドラインを使って再現されているため、デスク
トップ環境がなくてもプレイできます。ゲームプレイ：矢印キーで機体を操作し、スペースキーで発射します。
インストール：まずJavaをインストールしてください。次に、Releaseフォルダからクラスファイルをダウンロードします。ターミナルで`java FighterGame`と入力し
てゲームを起動してください。
対応OS：Linux、macOS、FreeBSD、OpenBSD、NetBSD、その他Unix系システム。
